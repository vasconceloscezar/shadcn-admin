# UI App

React + TypeScript frontend based on shadcn-admin template.

## Development

From the monorepo root:

```bash
# Start dev server
pnpm dev:ui

# Or run directly
cd apps/ui
pnpm dev
```

## Build

```bash
pnpm build:ui
```

## Tech Stack

- React 19
- TypeScript
- Vite
- TanStack Router
- TanStack Query
- shadcn/ui
- Tailwind CSS v4
